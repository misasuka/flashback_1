# 历史时间轴挑战（离线单页小游戏）

一个受 NYT Flashback 启发的中文历史小游戏：玩家需要将事件按时间顺序插入时间轴。

## 功能说明

- 保留核心玩法：给定“待放置事件”，点击时间轴插入位置。
- 初始随机给 1 个基准事件。
- 之后每次出现一个新事件，玩家选择插入位置。
- 放对 +1 分，放错 +0 分。
- 实时显示当前分数、已完成题数。
- 所有题目结束后显示结果页，并可一键重新开始。
- 题库为本地 `JSON` 文件，包含 30 个中国历史事件（1949 年以前）。

## 项目结构

```text
flashback_1/
├─ index.html              # 单页入口
├─ styles/
│  └─ main.css             # 页面样式
├─ js/
│  ├─ data.js              # 数据加载与清洗（优先 JSON，失败走离线回退）
│  ├─ events-fallback.js   # 离线回退题库（保障 file:// 打开可玩）
│  └─ game.js              # 游戏核心逻辑与渲染
└─ data/
   └─ events.json          # 30 道历史事件题库
```

## 数据格式

`data/events.json` 每条记录字段如下：

- `id`: 数字，唯一标识
- `title`: 事件标题
- `description`: 事件描述
- `date`: 日期字符串（推荐 `YYYY-MM-DD`）

示例：

```json
{
  "id": 17,
  "title": "武昌起义",
  "description": "辛亥革命首义爆发，清朝统治迅速瓦解。",
  "date": "1911-10-10"
}
```

## 本地运行

### 方式一：直接双击 `index.html`

- 直接在浏览器打开即可游玩。
- 某些浏览器对 `file://` 的 JSON 读取有限制，项目已提供 `events-fallback.js` 保证离线可玩。

### 方式二（推荐）：本地静态服务器

```bash
python -m http.server 8080
```

然后访问：`http://localhost:8080`
